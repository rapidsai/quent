# Quent Book

This directory contains the mdBook documentation for Quent.

## Prerequisites

Install [mdBook](https://rust-lang.github.io/mdBook/):

E.g.

```sh
cargo install mdbook
```

## Usage

Build and serve the book locally:

```sh
mdbook serve
```

Then open <http://localhost:3000> in your browser.

To just build the book:

```sh
mdbook build
```

The output will be in the `./book/` directory.
